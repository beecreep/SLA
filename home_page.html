TCC - Aplicação de Rede Local com Raspberry Pi

Este TCC desenvolve uma aplicação de rede local utilizando Raspberry Pi para facilitar o compartilhamento de atividades acadêmicas entre alunos e professores. A solução é acessível, segura e fácil de implementar, oferecendo uma plataforma colaborativa para gerenciar e trocar arquivos educacionais em instituições de ensino.

Sobre o Projeto

Este projeto de TCC utiliza um Raspberry Pi como servidor para uma aplicação de rede local, destinada ao compartilhamento de atividades e arquivos entre alunos e professores. A proposta é criar um ambiente colaborativo, acessível e de fácil manutenção.

Materiais Utilizados no Projeto

Para desenvolver este projeto de rede local com Raspberry Pi, utilizamos uma série de componentes e softwares essenciais. Abaixo estão os principais materiais utilizados:

Raspberry Pi

Um microcomputador de baixo custo que serve como o núcleo do sistema, permitindo o compartilhamento de dados e a configuração de uma rede local.

Cartão MicroSD

Usado para armazenar o sistema operacional e os dados do projeto. Um cartão de alta capacidade e boa velocidade é recomendado.

Fonte de Alimentação

Fornece energia para o Raspberry Pi. Uma fonte com saída de 5V e 3A é ideal para garantir o funcionamento estável do sistema.

Cabo Ethernet

Conecta o Raspberry Pi ao roteador para configurar a rede local. Um cabo de qualidade é importante para garantir uma boa conexão.

Roteador

O roteador é responsável por distribuir a conexão de rede entre os dispositivos conectados à rede local.

Software de Controle

Utilizamos software específico para o controle e monitoramento da rede, como Raspbian, Apache, e PHP para o servidor web.

O Coração do Projeto: Raspberry Pi

Para este projeto, estamos utilizando o Raspberry Pi 4, a versão mais poderosa e versátil da família Raspberry Pi. Com um processador ARM Cortex-A72 quad-core, até 8GB de RAM, e conectividade avançada, ele é o microcomputador ideal para criar uma rede local robusta e eficiente para o compartilhamento de atividades e arquivos entre alunos e professores.

Por que Escolhemos o Raspberry Pi 4?

  • Desempenho Superior: O Pi 4 oferece um desempenho até três vezes superior ao seu antecessor, o Raspberry Pi 3, permitindo o processamento rápido de dados e uma experiência de usuário suave.
  • Memória Ampliada: Com opções de até 8GB de RAM, o Pi 4 pode lidar com múltiplas tarefas simultâneas, desde o gerenciamento de arquivos até a execução de aplicações mais complexas.
  • Conectividade Moderna: Possui portas USB 3.0 para transferência rápida de dados, além de conectividade Wi-Fi e Bluetooth integradas para uma configuração sem complicações.
  • Suporte a 4K: Se for necessário, o Pi 4 pode ser conectado a monitores 4K, tornando-o perfeito para apresentações e visualização de conteúdos de alta definição.

Comparando com o Raspberry Pi 3

Embora o Raspberry Pi 3 ainda seja uma escolha sólida, o Pi 4 traz melhorias significativas em termos de desempenho e funcionalidades. Para um projeto que exige confiabilidade e capacidade de expansão, o Pi 4 é a escolha ideal, oferecendo mais poder de processamento e melhores opções de conectividade.

Aplicações no Projeto

No contexto deste TCC, o Raspberry Pi 4 será utilizado como o servidor principal da rede local, gerenciando o armazenamento e o compartilhamento de arquivos entre alunos e professores. Sua potência garante que o sistema seja rápido, seguro, e capaz de lidar com múltiplos usuários simultaneamente, sem comprometer o desempenho.

Estamos confiantes de que o Raspberry Pi 4 não só atende, mas excede as necessidades do nosso projeto, garantindo uma experiência de uso eficiente e moderna para todos os envolvidos.
