SLA - Sistema local academico

Este TCC desenvolve uma aplicação de rede local utilizando Raspberry Pi para facilitar o compartilhamento de atividades acadêmicas entre alunos e professores. A solução é acessível, segura e fácil de implementar, oferecendo uma plataforma colaborativa para gerenciar e trocar arquivos educacionais em instituições de ensino.

Sobre o Projeto

Este projeto de TCC utiliza um Raspberry Pi como servidor para uma aplicação de rede local, destinada ao compartilhamento de atividades e arquivos entre alunos e professores. A proposta é criar um ambiente colaborativo, acessível e de fácil manutenção.

Materiais Utilizados no Projeto

Para desenvolver este projeto de rede local com Raspberry Pi, utilizamos uma série de componentes e softwares essenciais. Abaixo estão os principais materiais utilizados:

Raspberry Pi

Um microcomputador de baixo custo que serve como o núcleo do sistema, permitindo o compartilhamento de dados e a configuração de uma rede local.

Cartão MicroSD

Usado para armazenar o sistema operacional e os dados do projeto. Um cartão de alta capacidade e boa velocidade é recomendado.

Fonte de Alimentação

Fornece energia para o Raspberry Pi. Uma fonte com saída de 5V e 3A é ideal para garantir o funcionamento estável do sistema.

Cabo Ethernet

Conecta o Raspberry Pi ao roteador para configurar a rede local. Um cabo de qualidade é importante para garantir uma boa conexão.

Roteador

O roteador é responsável por distribuir a conexão de rede entre os dispositivos conectados à rede local.

Software de Controle

Utilizamos software específico para o controle e monitoramento da rede, como Raspbian, Apache, e PHP para o servidor web.

O Coração do Projeto: Raspberry Pi 3

O Raspberry Pi 3 é um microcomputador compacto e eficiente, amplamente utilizado em projetos educacionais, IoT e de automação. Ele combina poder de processamento, conectividade integrada e baixo custo, tornando-se a escolha ideal para criar a infraestrutura necessária ao nosso projeto.

Principais Características do Raspberry Pi 3

  • Processador Quad-Core: Equipado com um processador ARM Cortex-A53 de 1.2GHz, o Raspberry Pi 3 é capaz de realizar múltiplas tarefas simultaneamente, como gerenciar conexões de usuários e processar solicitações de dados.
  • Memória RAM: Possui 1GB de memória RAM, suficiente para suportar aplicações de médio porte, como servidores locais e interfaces web interativas.
  • Conectividade Integrada: Inclui Wi-Fi 802.11n e Bluetooth 4.1, eliminando a necessidade de adaptadores externos para redes sem fio ou dispositivos periféricos.
  • Portas USB e GPIO: Conta com 4 portas USB e 40 pinos GPIO (General Purpose Input/Output), permitindo a conexão de diversos dispositivos e sensores, caso o projeto seja expandido no futuro.
  • Armazenamento com microSD: Utiliza cartões microSD para armazenamento do sistema operacional e dados, oferecendo flexibilidade e fácil substituição em caso de necessidade.
  • Eficiência Energética: Consome pouca energia (5V via micro-USB), tornando-o ideal para uso contínuo em ambientes educacionais ou de trabalho colaborativo.

Por que Escolhemos o Raspberry Pi 3?

O Raspberry Pi 3 oferece o equilíbrio perfeito entre custo, desempenho e simplicidade. Embora existam versões mais avançadas, como o Raspberry Pi 4, as necessidades do nosso projeto são atendidas com folga pelo modelo 3. Além disso, sua compatibilidade com uma vasta gama de softwares e sistemas operacionais facilita o desenvolvimento e a manutenção.

Como o Raspberry Pi 3 Contribui para o Projeto?

  • Servidor Local: O Raspberry Pi 3 é o núcleo do sistema, atuando como servidor para armazenar e gerenciar as atividades e respostas compartilhadas entre alunos e professores.
  • Acessibilidade: Graças ao Wi-Fi integrado, ele permite que dispositivos na mesma rede local acessem o sistema sem a necessidade de conexões físicas.
  • Confiabilidade: O Raspberry Pi 3 é conhecido por sua estabilidade, garantindo que o sistema funcione de maneira ininterrupta mesmo com múltiplos usuários simultâneos.

Mais Aplicações do Raspberry Pi 3

Além do uso como servidor, o Raspberry Pi 3 pode ser expandido para diversas outras aplicações, como:

  • Streaming de Conteúdo: Pode ser usado para compartilhar vídeos ou apresentações durante aulas.
  • Automação: Com sensores e atuadores conectados aos pinos GPIO, é possível integrar funcionalidades como monitoramento de ambiente.
  • Aprendizado de Programação: Ideal para ensinar lógica de programação, desenvolvimento web e automação em ambientes educacionais.

Conclusão

O Raspberry Pi 3 não é apenas uma escolha prática, mas estratégica para o nosso projeto. Ele oferece a base sólida necessária para criar um sistema eficiente, econômico e adaptável às necessidades de professores e alunos. Sua popularidade na comunidade de tecnologia garante suporte abundante, tornando-o a ferramenta ideal para transformar ideias em realidade.
